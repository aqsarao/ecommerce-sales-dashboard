# E-Commerce Sales Analytics Dashboard

# Project Overview
This project analyzes e-commerce sales data and provides insights using data analysis, visualization, and machine learning.

# Features
- Data Cleaning & Feature Engineering
- Exploratory Data Analysis (EDA)
- Interactive Dashboard using Streamlit
- Sales Prediction using Machine Learning

# Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Streamlit
- Scikit-learn

# How to Run
```bash
streamlit run app.py
